Why SysLens?

Direct /proc Parsing

No intermediate libraries. SysLens reads straight from the Linux kernel’s virtual filesystem for maximum speed and accuracy.

Low Resource Footprint

Written in optimized C99. It consumes less than 1MB of memory, making it ideal for servers and embedded systems.

Visual Intelligence

Dynamic progress bars with color-coded alerts (Green/Yellow/Red) help you identify bottlenecks at a single glance.

Installation Guide

# 1. Clone the repository

git clone https://github.com/mahmudul626/syslens.git

cd syslens

# 2. Compile from source

make

# 3. Install globally (optional)

sudo make install

Prerequisites

  • Linux OS (Kernel 2.6+)
  • GCC Compiler
  • Make Utility

Uninstallation

To remove the binary cleanly:

sudo make uninstall

Usage & Flags

syslens -s

System Info Only

Displays OS, Kernel version, CPU model, and current uptime.

syslens -m

Memory & Load

Focuses on RAM, Swap, and System Load averages (1/5/15 min).

syslens -p

Process Analytics

Shows active task summary: Running, Sleeping, and Zombies.

syslens --help

Help Menu

Shows all available commands and version information.
